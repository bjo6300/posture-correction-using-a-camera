# Posture-Correction-using-a-camera-Program

카메라를 이용한 자세교정 프로그램

## Project Introduction
카메라를 이용하여 촬영한 영상 이미지를 분석해 잘못된 자세를 분석 및 교정하는 시스템입니다. 잘못된 자세로는 거북목, 턱 괴는 자세, 어깨비대칭이 있습니다.
